Project Introduction

Graduation Project


- Using slam technology to create a map by attaching a lidar sensor and a camera to the robot and combining the surrounding information obtained from the sensor while moving around.


- Based on the saved map, it detects the current location of the robot, drives autonomously when a person designates a destination, and avoids collisions until it arrives at the destination.
